#### 工厂模式

在父类中提供一个创建对象的方法，允许子类决定实例化对象类型

```python
dog = PatFactory.getPat('dog')
dog.fucked()
```

#### 装饰器模式

不改变原有类基础上给类增加新功能

```python
@decorater
def func:
  pass
# -------------
decorater(dog).fucked()
```
